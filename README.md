# igdb-types

TypeScript type definitions for IGDB API responses.

## Installation

```bash
npm install igdb-types
```

## Usage

```typescript
import { Game, Company } from 'igdb-types';

// Use the types in your code
const game: Game = {
  // ... game properties
};
```

## Development

To install dependencies:

```bash
npm install
```

To build the package:

```bash
npm run build
```
